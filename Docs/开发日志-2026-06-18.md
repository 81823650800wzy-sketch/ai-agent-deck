## 完成的工作

### 1. BLE 设备通信管理器

**文件**: `Manager/device_manager.py`

重写 `DeviceManager` 类，支持 BLE GATT 通信：

```python
class DeviceManager:
    # 功能:
    # - 自动扫描 "AI Agent Deck" 设备
    # - BLE GATT 连接
    # - 心跳保活 (5秒间隔)
    # - 断线自动重连 (指数退避)
    # - 分块发送 (MTU 限制)
```

**核心特性**:
- **自动扫描**: 使用 bleak 库扫描 BLE 设备
- **GATT 通信**: 通过 128-bit UUID 与 ESP32 通信
- **心跳保活**: 每 5 秒发送 ping，检测连接状态
- **断线重连**: 指数退避策略，最大延迟 30 秒
- **线程安全**: 后台线程运行异步事件循环

**新增备用类**: `DeviceManagerSerial`
- 当 BLE 不可用时使用串口通信
- 保持向后兼容

### 2. WorkflowManager 更新

**文件**: `Manager/workflow_manager.py`

更新初始化参数，支持选择通信方式：

```python
class WorkflowManager:
    def __init__(self, use_ble: bool = True, com_port: str = None):
        if use_ble:
            self.device = DeviceManager()  # BLE
        else:
            self.device = DeviceManagerSerial(port=com_port)  # 串口
```

### 3. main.py 参数更新

新增命令行参数：
- `--serial`: 使用串口通信
- `--port`: 指定串口号

```bash
python main.py              # 默认 BLE
python main.py --serial     # 串口模式
python main.py --port COM3  # 指定串口
python main.py --scan       # 扫描 BLE 设备
```
