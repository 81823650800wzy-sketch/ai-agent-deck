| **GPIO 8** | **RST** | 复位（低电平有效） |
| **GPIO 21** | **BL** | 背光控制 |
| **3.3V** | **VCC** | 电源正极 |
| **GND** | **GND** | 电源地 |

## 2. 按键矩阵接线 (3×2)

**矩阵结构**：
```
        COL1 (GPIO7)    COL2 (GPIO15)
           │                │
ROW1(GPIO4)──┤K1├──────────┤K2├──
           │                │
ROW2(GPIO5)──┤K3├──────────┤K4├──
           │                │
ROW3(GPIO6)──┤K5├──────────┤K6├──
```

**每个按键连接**：
```
按键一端 → ROW 引脚
按键另一端 → COL 引脚
```

**具体接线**：

| 按键 | ROW 引脚 | COL 引脚 | 功能 |
|------|----------|----------|------|
| K1 | GPIO 4 | GPIO 7 | GPT |
| K2 | GPIO 4 | GPIO 15 | Claude |
| K3 | GPIO 5 | GPIO 7 | Cursor |
| K4 | GPIO 5 | GPIO 15 | Voice |
| K5 | GPIO 6 | GPIO 7 | OCR |
| K6 | GPIO 6 | GPIO 15 | Custom |

**接线图**：
```
ESP32-S3 DevKitC              机械轴 (热插拔座)
┌─────────────────┐          ┌──────────────┐
│               4 ├──────────┤ K1 ROW 端    │
│               4 ├──────────┤ K2 ROW 端    │
│               5 ├──────────┤ K3 ROW 端    │
│               5 ├──────────┤ K4 ROW 端    │
│               6 ├──────────┤ K5 ROW 端    │
│               6 ├──────────┤ K6 ROW 端    │
│               7 ├──────────┤ K1 COL 端    │
│               7 ├──────────┤ K3 COL 端    │
│               7 ├──────────┤ K5 COL 端    │
│              15 ├──────────┤ K2 COL 端    │
│              15 ├──────────┤ K4 COL 端    │
│              15 ├──────────┤ K6 COL 端    │
└─────────────────┘          └──────────────┘
```

**注意**：
- 矩阵中不需要二极管（3×2 矩阵不会产生鬼键）
- 热插拔座两端不分方向
- 按键按下时，ROW 和 COL 导通

## 3. 摇杆接线

| ESP32-S3 DevKitC | ALPS 摇杆 | 说明 |
|------------------|-----------|------|
| **3.3V** | **VCC** | 电源正极 |
| **GND** | **GND** | 电源地 |
| **GPIO 1** | **VRx** | X 轴模拟输出 (ADC1_CH0) |
| **GPIO 2** | **VRy** | Y 轴模拟输出 (ADC1_CH1) |
| **GPIO 3** | **SW** | 按键开关（按下接地） |

```
ESP32-S3 DevKitC              ALPS 摇杆
┌─────────────────┐          ┌──────────────┐
│             3V3 ├──────────┤ VCC          │
│             GND ├──────────┤ GND          │
│               1 ├──────────┤ VRx          │
│               2 ├──────────┤ VRy          │
│               3 ├──────────┤ SW           │
└─────────────────┘          └──────────────┘
```

**摇杆引脚说明**：
```
        ┌─────────────┐
        │  ALPS 摇杆   │
        │             │
   VCC ─┤  ●          │
   VRx ─┤  ●    ●  ─ SW
   VRy ─┤  ●          │
   GND ─┤  ●          │
        └─────────────┘
```

**注意**：
- VRx/VRy 输出 0-3.3V 模拟电压
- SW 按下时接地，GPIO3 需要上拉
- 建议在 VRx/VRy 和 GND 之间加 100nF 滤波电容（可选）

## 4. 完整接线汇总

### 所有 GPIO 使用

| GPIO | 功能 | 类型 |
|------|------|------|
| GPIO 1 | 摇杆 VRx | ADC 输入 |
| GPIO 2 | 摇杆 VRy | ADC 输入 |
| GPIO 3 | 摇杆 SW | 数字输入（上拉） |
| GPIO 4 | 矩阵 ROW1 | 输出 |
| GPIO 5 | 矩阵 ROW2 | 输出 |
| GPIO 6 | 矩阵 ROW3 | 输出 |
| GPIO 7 | 矩阵 COL1 | 输入（上拉） |
| GPIO 8 | ST7789 RST | 输出 |
| GPIO 9 | ST7789 DC | 输出 |
| GPIO 10 | ST7789 CS | 输出 |
| GPIO 11 | ST7789 MOSI | SPI 输出 |
| GPIO 12 | ST7789 SCLK | SPI 输出 |
| GPIO 15 | 矩阵 COL2 | 输入（上拉） |
| GPIO 21 | ST7789 BL | 输出 |

### 电源连接

| 电源 | 连接 |
|------|------|
| 3.3V | ST7789 VCC, 摇杆 VCC |
| GND | ST7789 GND, 摇杆 GND |

## 6. 常见接线问题

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| 屏幕不亮 | BL 未接 | 检查 GPIO21→BL |
| 白屏 | MOSI/SCLK 接反 | 检查 GPIO11→MOSI, GPIO12→SCLK |
| 颜色异常 | DC 接错 | 检查 GPIO9→DC |
| 按键无反应 | ROW/COL 接错 | 按矩阵图重新检查 |
| 摇杆值不变 | VRx/VRy 接错 | 检查 GPIO1→VRx, GPIO2→VRy |
| 摇杆按键无反应 | SW 未接 | 检查 GPIO3→SW |

## 7. 测试顺序建议

1. **先接显示屏** → 烧录显示测试固件 → 确认屏幕正常
2. **再接摇杆** → 烧录输入测试固件 → 确认摇杆正常
3. **最后接按键** → 烧录输入测试固件 → 确认按键正常

每接一个模块就测试一次，不要一次性全部接上。
