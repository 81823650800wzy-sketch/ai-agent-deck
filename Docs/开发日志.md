### 构建环境修复

**问题**：Python 依赖版本不匹配
```
esptool: 需要 ~=4.12.dev1，已安装 5.3.0
cryptography: 需要 <45，已安装 49.0.0
esp-idf-kconfig: 需要 <3.0.0，已安装 3.11.1
```

**解决**：使用 ESP-IDF 约束文件重装 Python 依赖
```bash
pip install --force-reinstall -c D:\Espressif\espidf.constraints.v5.5.txt \
    -r D:\Espressif\frameworks\esp-idf-v5.5.4\tools\requirements\requirements.core.txt
```

**结果**：esptool=4.12.dev2, cryptography=44.0.3 ✅
