# AI Agent Deck V1.0 开发规划

> **适用对象**：第一次做硬件开发的个人开发者
> **开发方式**：借助AI工具（Claude/Cursor/GPT）辅助开发
> **预计周期**：2个月（8周）
> **总预算**：约600-800元

---

## 核心原则

### 渐进式验证

**绝对不要**一次性买齐所有元件然后全部焊上。

正确做法：
```
开发板验证 → Mini PCB → 正式PCB
```

每一阶段验证通过后再进入下一阶段。

### 模块化思维

将整个系统拆分为独立模块：
- 电源模块
- 主控模块
- 显示模块
- 输入模块
- 扩展模块

每个模块独立验证，最后整合。

---

## 开发阶段

### Phase 1：开发板验证（第1-2周）

**目标**：用开发板验证所有核心功能

**所需材料**：
- ESP32-S3 DevKitC ×2 ✅
- ST7789 IPS模块 ×2 ✅
- ALPS摇杆 ×2 ✅
- 机械轴 + 热插拔座 + 键帽 ✅

**验证内容**：
1. ESP32 最小系统（串口通信、LED闪烁）
2. ST7789 显示（颜色测试、文字显示）
3. 按键输入（单键检测、矩阵扫描、消抖）
4. 摇杆 ADC（静止值、死区、四方向）
5. BLE HID（PC配对、发送按键、组合键）

### Phase 2：功能集成（第3-4周）

**目标**：将所有模块整合，实现完整功能

1. UI 系统（主界面、菜单导航）
2. 按键映射（GPT/Claude/Cursor快捷键）
3. 电池管理（电压检测、电量百分比）
4. 整体调试（内存、任务优先级、稳定性）

### Phase 3：PCB设计（第5-6周）

**目标**：设计并验证PCB

1. 原理图设计（嘉立创EDA）
2. PCB Layout（4层板、天线净空）
3. 打样验证

### Phase 4：焊接调试（第7-8周）

**目标**：焊接PCB并完成整机调试
