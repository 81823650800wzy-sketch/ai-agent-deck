# AI Agent Deck V2.0 开发规划

> Context-Aware Workflow Controller
> 目标：自动感知用户工作场景，自动适配工作流，降低软件间切换记忆负担
> 最终交付：个人工作流数据库

---

## 核心理念

```
软件识别 → 工作流识别 → 自动适配 → 行为记录 → 个人数据库
                                                      ↓
                                                AI 分析推荐
```

## 模块清单

### 模块 1：设备管理（BLE 连接生命周期）

- 自动扫描 BLE 设备
- 自动连接 "AI Agent Deck"
- 握手协议（设备信息、固件版本、当前 Profile）
- 双向通道（HID + GATT）
- 心跳保活（5 秒 ping，断线自动重连）

```python
class DeviceManager:
    async def scan() -> list[DeviceInfo]
    async def connect(device: DeviceInfo) -> bool
    async def disconnect()
    async def send_profile(profile: Profile)
    async def send_config(config: DeviceConfig)
    async def on_key_press(callback)
    async def on_device_event(callback)
    def is_connected() -> bool
    def battery_level() -> int
```

### 模块 2：软件识别（窗口检测 + 进程分析）

- 每 300ms 检测前台窗口
- 提取：进程名、窗口标题、进程路径、窗口类名
- 映射到标准化应用标识

```python
@dataclass
class AppInfo:
    process_name: str       # Code.exe
    window_title: str       # main.py - VSCode
    process_path: str       # C:\...\Code.exe
    app_id: str             # vscode
    timestamp: float

class WindowDetector:
    def get_active_app() -> AppInfo
    def on_app_changed(callback)
```

### 模块 3：Profile 系统（按键映射 + 动态配置）

- 支持多层 Profile（默认层 + 调试层 + 自定义层）
- 支持上下文感知（同一软件不同场景不同映射）
- 支持导入导出

```python
class ProfileManager:
    def load_profiles() -> list[Profile]
    def get_profile(app_id: str) -> Profile
    def save_profile(profile: Profile)
    def update_key(profile_id, key_id, mapping)
    def export_profile(profile_id) -> dict
    def import_profile(data) -> Profile
```

### 模块 4：行为记录（使用日志 + SQLite）

```python
@dataclass
class BehaviorRecord:
    timestamp: float
    event_type: str         # app_switch | key_press | key_customize
    app_id: str
    app_name: str
    window_title: str
    key_id: str | None
    key_action: str | None
    session_id: str
    context: dict

class BehaviorLogger:
    def log_app_switch(app: AppInfo)
    def log_key_press(app: AppInfo, key: KeyMapping)
    def log_customization(app, key, old, new)
    def get_history(start, end) -> list[BehaviorRecord]
    def get_app_stats(app_id) -> AppStats
    def get_daily_summary(date) -> DailySummary
```

### 模块 5：工作流分析（模式识别）

- 软件切换频率矩阵
- 按键使用频率统计
- 高频使用时段分析
- 工作流模式识别

```python
class WorkflowAnalyzer:
    def analyze_sessions(date_range) -> list[WorkflowSession]
    def get_app_usage_stats(app_id) -> UsageStats
    def get_workflow_patterns() -> list[WorkflowPattern]
    def get_switch_matrix() -> dict

```

### 模块 6：AI 推荐引擎（预留接口）

```python
class AIRecommender:
    def analyze_usage(logs) -> UsageAnalysis
```

## 技术选型

| 组件 | 选型 | 理由 |
|------|------|------|
| PC 通信 | 串口 + BLE | 串口调试稳定，BLE 用于产品形态 |
| 数据存储 | SQLite | 轻量、单文件、免服务 |
| GUI 框架 | tkinter / PyQt | Python 原生，快速开发 |
| 行为分析 | Python + pandas | 数据处理成熟 |
| AI 推荐 | 规则引擎 + LLM API | 先本地规则，后接云端 LLM |
| ESP32 UI | 自绘 (ST7789) | 已有基础，可控性强 |
