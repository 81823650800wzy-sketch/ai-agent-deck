## Step 2: ST7789 显示屏验证 🔄

**目标**：确认屏幕能正常显示

**接线**：
```
ESP32-S3 DevKitC    →    ST7789 模块
─────────────────────────────────────
GPIO 11 (MOSI)      →    MOSI
GPIO 12 (SCLK)      →    SCLK
GPIO 10 (CS)         →    CS
GPIO 9  (DC)         →    DC
GPIO 8  (RST)        →    RST
GPIO 21 (BL)         →    BL
3.3V                 →    VCC
GND                  →    GND
```

**烧录**：
```bash
cd D:\AI-Agent-Deck\Firmware
bash build.sh flash COMx    # 替换 COMx 为实际端口
```

**验证**：
- [ ] 串口输出正常（能看到 "AI Agent Deck V1.0" 日志）
- [ ] 屏幕点亮
- [ ] 颜色条测试通过（8种颜色正确显示）
- [ ] 棋盘格测试通过（黑白格子清晰）
- [ ] 渐变测试通过（色彩过渡平滑）
- [ ] 无坏点
- [ ] 无闪烁

**常见问题**：

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| 屏幕不亮 | 背光未接 | 检查 BL→GPIO21 接线 |
| 白屏 | SPI 接线错误 | 检查 MOSI/SCLK/CS 接线 |
| 颜色异常 | DC 接线错误 | 检查 DC→GPIO9 接线 |
| 乱码 | RST 未接 | 检查 RST→GPIO8 接线 |

## Step 4: 3×2 按键矩阵扫描 ⬜

**目标**：确认矩阵扫描正确

**接线**：
```
ROW1 → GPIO4     COL1 → GPIO7
ROW2 → GPIO5     COL2 → GPIO15
ROW3 → GPIO6
```

**验证**：
- [ ] 6个按键都能正确识别
- [ ] 无鬼键
- [ ] 消抖正常（5ms）

## Step 5: 摇杆 ADC 读取 ⬜

**目标**：确认摇杆模拟输入正常

**接线**：
```
摇杆VCC → 3.3V
摇杆GND → GND
摇杆VRx → GPIO1（ADC）
摇杆VRy → GPIO2（ADC）
摇杆SW → GPIO3（上拉输入）
```

**验证**：
- [ ] 静止时 ADC 值约 2048
- [ ] 波动范围 ±50 以内
