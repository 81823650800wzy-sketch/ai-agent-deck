## 关键配置

### 1. MADCTL 寄存器 (0x36)

**正确的值: `0x40`**

```c
// 竖向模式
uint8_t madctl = 0x40;  // MX 位 (水平镜像)
send_cmd_data(0x36, &madctl, 1);
```

**位定义:**
| Bit | 名称 | 说明 |
|-----|------|------|
| 7 | MY | 行地址顺序 (0=正常) |
| 6 | MX | 列地址顺序 (1=水平镜像) ← **需要设置** |
| 5 | MV | 行/列交换 (0=竖向) |
| 4 | ML | 垂直刷新顺序 |
| 3 | RGB/BGR | 颜色顺序 (0=RGB) |
| 2 | MH | 水平刷新顺序 |

**不同方向的配置:**
```c
case 0: m=0x40; break;  /* 竖向 + MX */
case 1: m=0x70; break;  /* 横屏 + MX + MV */
case 2: m=0xD0; break;  /* 竖向翻转 + MX + MY */
case 3: m=0xB0; break;  /* 横屏翻转 + MX + MY + MV */
```

### 2. 像素格式 (0x3A)

```c
uint8_t colmod = 0x55;  // 16-bit/pixel (RGB565)
send_cmd_data(0x3A, &colmod, 1);
```

### 3. SPI 配置

```c
spi_clock_hz = 40 * 1000 * 1000;  // 40MHz
SPI mode = 0;
```

### 4. GRAM 偏移

```c
#define ROW_OFFSET  80    // 1.54寸屏幕的行偏移
#define COL_OFFSET  0     // 列偏移
```

## 字体绘制

### 5x7 点阵字体

**数据格式:**
- 每个字符 5 字节 (5列)
- 每字节低 7 位表示 7 行
- bit0 = 最上面像素, bit6 = 最下面像素

**绘制函数:**
```c
void draw_char(int16_t x, int16_t y, char c, uint16_t color, uint8_t scale)
{
    if (c < 32 || c > 127) c = '?';
    const uint8_t *glyph = font5x7[c - 32];

    for (int col = 0; col < 5; col++) {
        uint8_t line = glyph[col];
        for (int row = 0; row < 7; row++) {
            if (line & (1 << row)) {
                for (int sy = 0; sy < scale; sy++) {
                    for (int sx = 0; sx < scale; sx++) {
                        int16_t px = x + col * scale + sx;
                        int16_t py = y + row * scale + sy;
                        if (px >= 0 && px < LCD_W && py >= 0 && py < LCD_H) {
                            st7789_draw_pixel(px, py, color);
                        }
                    }
                }
            }
        }
    }
}
```

**字符串绘制:**
```c
void draw_string(int16_t x, int16_t y, const char *str, uint16_t color, uint8_t scale)
{
    while (*str) {
        draw_char(x, y, *str, color, scale);
        x += 6 * scale;  // 5像素宽 + 1像素间距
        str++;
    }
}

// 居中绘制
void draw_string_centered(int16_t y, const char *str, uint16_t color, uint8_t scale)
{
    int16_t len = 0;
    const char *p = str;
    while (*p) { len++; p++; }
    int16_t w = len * 6 * scale;
    int16_t x = (LCD_W - w) / 2;
    draw_string(x, y, str, color, scale);
}
```

## 颜色工具

### RGB565 宏
```c
#define RGB565(r,g,b) ((((r)&0xF8)<<8)|(((g)&0xFC)<<3)|((b)>>3))
```

### 颜色插值
```c
uint16_t color_lerp(uint16_t c1, uint16_t c2, uint8_t t)
{
    uint8_t r1 = (c1 >> 11) & 0x1F, g1 = (c1 >> 5) & 0x3F, b1 = c1 & 0x1F;
    uint8_t r2 = (c2 >> 11) & 0x1F, g2 = (c2 >> 5) & 0x3F, b2 = c2 & 0x1F;
    uint8_t r = r1 + ((int16_t)(r2 - r1) * t) / 255;
    uint8_t g = g1 + ((int16_t)(g2 - g1) * t) / 255;
    uint8_t b = b1 + ((int16_t)(b2 - b1) * t) / 255;
    return (r << 11) | (g << 5) | b;
}
```

### 渐变绘制 (带边界检查)
```c
void draw_gradient_h(int16_t x, int16_t y, int16_t w, int16_t h,
                     uint16_t c1, uint16_t c2)
{
    if (x < 0) { w += x; x = 0; }
    if (y < 0) { h += y; y = 0; }
    if (x + w > LCD_W) w = LCD_W - x;
    if (y + h > LCD_H) h = LCD_H - y;
    if (w <= 0 || h <= 0) return;

    for (int16_t i = 0; i < w; i++) {
        uint8_t t = (w > 1) ? (i * 255) / (w - 1) : 0;
        st7789_fill_rect(x + i, y, x + i, y + h - 1, color_lerp(c1, c2, t));
    }
}
```

## 常见问题

| 现象 | 原因 | 解决方案 |
|------|------|----------|
| 文字镜像 | MX 位未设置 | MADCTL = 0x40 |
| 颜色异常 (蓝↔红) | RGB/BGR 位错误 | MADCTL bit3 = 0 (RGB) |
| 显示偏移 | GRAM 偏移错误 | ROW_OFFSET = 80 |
| 屏幕不亮 | 背光未接 | BL → GPIO21 |
| 白屏 | SPI 接线错误 | 检查 MOSI/SCLK/CS |

## 复用代码模板

```c
/* ST7789 初始化模板 */
st7789_config_t lcd_cfg = {
    .spi_host = SPI3_HOST,
    .pin_mosi = GPIO_NUM_11,
    .pin_sclk = GPIO_NUM_12,
    .pin_cs = GPIO_NUM_10,
    .pin_dc = GPIO_NUM_9,
    .pin_rst = GPIO_NUM_8,
    .pin_bl = GPIO_NUM_21,
    .spi_clock_hz = 40 * 1000 * 1000,
};
st7789_init(&lcd_cfg);
st7789_backlight(true);

/* 设置方向 (竖向) */

st7789_set_madctl(madctl);
```
